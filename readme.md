# HTML & CSS Mini Projects

This folder contains a collection of small HTML and CSS mini projects created to practice and strengthen my frontend fundamentals.

## Purpose
- Practice building real-world UI layouts using pure HTML and CSS
- Improve understanding of CSS layout techniques
- Strengthen visual structure, spacing, and alignment skills

## Focus Areas
- CSS Flexbox for one-dimensional layouts
- CSS Grid for complex and responsive layouts
- Page structure and layout planning
- Responsive design fundamentals
- Clean and reusable CSS

## Learning Approach
- Projects are built from scratch without using frameworks
- Focus on UI cloning and layout accuracy
- Each project represents a small, focused learning exercise

## Tech Stack
- HTML5
- CSS3
